# 🚀 AI-Powered CRM System (MERN Stack)

🔗 **Live Demo:** https://crm-project-frontend-xi.vercel.app/

---

## 📌 Overview

This is a full-stack Customer Relationship Management (CRM) system built using the MERN stack. The application manages the complete lead lifecycle — from lead generation to conversion — with secure authentication, AI-assisted insights, and integrated online payments.

---

## ✨ Features

- 🔐 JWT Authentication & Role-Based Access Control (RBAC)
- 📊 Lead Management System (Create, Update, Assign, Track)
- 🔄 Sales Pipeline (Kanban Workflow)
- 🤖 AI-Assisted Lead Prioritization & Smart Insights
- 📅 Follow-Up Reminders & Activity Tracking
- 💳 Payment Integration (Cashfree Gateway)
- 📈 Dashboard with Analytics & Reports
- 🌐 Fully Responsive UI (Mobile + Desktop)

---

  - Admin

  - Sales Executive

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository
```bash
git clone https://github.com/Navinraj2405/CRM-project.git
cd CRM-project
```

### 2️⃣ Setup Frontend
```bash
cd Frontend
npm install
npm run dev
```

### 3️⃣ Setup Backend
```bash
cd Backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in Backend:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key

CASHFREE_APP_ID=your_app_id
CASHFREE_SECRET_KEY=your_secret_key
```
